# 🏥 Hospital Management System

A desktop-based Hospital Management System developed using **Java, JDBC, and MySQL** to simplify hospital administration by managing patients, doctors, departments, rooms, appointments, and billing information.

---

## 📌 Features

* 👨‍⚕️ Patient Registration
* 🏥 Department Management
* 🛏️ Room Management
* 🚑 Ambulance Information
* 📅 Patient Admission & Discharge
* 👩‍💼 Reception Dashboard
* 🔐 Login System
* 📝 Update Patient Details
* 🔍 Search Room Availability
* 💾 MySQL Database Connectivity using JDBC

---

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/shubham-sharma-03/Hospital-Management-System.git
```

2. Create a MySQL database.

3. Update the database credentials inside `conn.java`.

4. Import the SQL database.

5. Run `Login.java`.
